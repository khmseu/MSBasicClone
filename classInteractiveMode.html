Detailed Description

Interactive REPL for Applesoft BASIC.

Provides command-line interface for direct interaction with the interpreter. Accepts immediate commands and program line entry.

Usage:

InteractiveMode repl;
repl.run(); // Enters REPL until user exits

void InteractiveMode::run ( )

Enter REPL loop.

Run the interactive REPL loop.

Displays banner, then repeatedly prompts for input and executes commands until user exits (Ctrl+C or EXIT command).

Displays the banner, creates an interpreter, and enters the main loop:

  1. Display prompt
  2. Read input line
  3. Execute line (immediate command or program line entry)
  4. Display errors with "?" prefix
  5. Repeat until EOF (Ctrl+D on Unix, Ctrl+Z on Windows)
