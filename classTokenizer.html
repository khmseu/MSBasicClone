Detailed Description

Converts BASIC source code into a stream of tokens.

The Tokenizer class performs lexical analysis on BASIC source code, breaking it down into tokens that the parser can process. It handles all Applesoft BASIC syntax including keywords, operators, literals, and identifiers.

Usage:

Tokenizer tokenizer;
std::vector<Token> tokens = tokenizer.tokenize("PRINT \"HELLO\"");

TokenType Tokenizer::getKeywordType ( const std::string &  word) const

Get the token type for a keyword.

Resolve a keyword string to its TokenType.

Parameters
wordThe keyword string
Returns
TokenType The corresponding token type
Exceptions
std::runtime_errorif word is not a keyword

Performs case-insensitive lookup in the tokenizer's keyword table and returns the corresponding TokenType.

Details:

  • The keyword map includes commands (PRINT, INPUT, LIST, RUN, NEW), control flow (IF, THEN, ELSE, FOR, NEXT, GOTO, GOSUB), data keywords (DATA, READ, RESTORE, DIM), functions (FN, DEF), graphics (GR, HGR, HPLOT, DRAW, XDRAW), and ProDOS commands (OPEN, CLOSE, READ, WRITE).
  • Special tokens like "?" (alias for PRINT) and multi-character tokens such as "COLOR=" and "HCOLOR=" are included.
  • Keywords are matched case-insensitively; input is converted to uppercase.

Notes:

  • If the word is not recognized as a keyword, returns TokenType::IDENTIFIER.
Parameters
wordInput word to classify (case-insensitive)
Returns
TokenType for the keyword, or TokenType::IDENTIFIER if not found
