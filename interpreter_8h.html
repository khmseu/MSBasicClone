Detailed Description

Runtime execution engine for Applesoft BASIC programs.

The Interpreter class is the core runtime engine that executes parsed BASIC programs. It manages program state, variable storage, control flow, I/O operations, and graphics modes while maintaining Applesoft BASIC compatibility.

Key responsibilities:

  • Program lifecycle (NEW, LOAD, SAVE, RUN, LIST)
  • Statement execution and control flow (GOTO, GOSUB, FOR/NEXT, IF/THEN)
  • Variable and array management via Variables class
  • DATA/READ/RESTORE sequence management
  • I/O operations (PRINT, INPUT, GET)
  • Graphics mode control (GR, HGR, HPLOT, COLOR, etc.)
  • ProDOS-compatible file operations
  • Error handling with Applesoft error messages
  • TRACE/NOTRACE execution tracing
  • SPEED command for statement delays

The interpreter maintains program lines in sorted order and supports both immediate mode (direct command execution) and program mode (running stored programs). It handles the full Applesoft BASIC statement set including DEF FN user-defined functions, ONERR error trapping, and CHAIN/- program loading.
