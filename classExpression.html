Detailed Description

Base class for all expression AST nodes.

Expression is an abstract base class representing any expression that can be evaluated to produce a Value. Concrete subclasses implement specific expression types (literals, variables, binary operations, etc.).

All expressions must implement the evaluate() method which computes the expression's value given an Interpreter context.
