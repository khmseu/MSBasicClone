Detailed Description

Applesoft-compatible 40-bit floating point number.

Implements the 5-byte floating point format used by Applesoft BASIC, providing arithmetic operations and math functions with precision matching the original Apple II implementation.

Usage:

Float40 a(3.14159);
Float40 b("2.71828");
Float40 result = a * b;
std::cout << result.toString() << std::endl;
