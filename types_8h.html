Detailed Description

Core type definitions for the MSBasic interpreter.

This file defines the fundamental types used throughout the MSBasic interpreter, including token types, values, program structures, and AST node types. It serves as the central type system for the entire interpreter.

Enumeration Type Documentation

◆ TokenType

enum class TokenType
strong

Enumeration of all token types recognized by the tokenizer.

This enum defines all possible token types that can appear in BASIC code, including literals, keywords, operators, built-in functions, and delimiters. The tokenizer converts raw text into a stream of these typed tokens.
