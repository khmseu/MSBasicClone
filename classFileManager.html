Detailed Description

Singleton managing open file handles for BASIC file I/O.

Provides ProDOS-style file operations with integer handles. Maintains open file state and coordinates file access across BASIC commands.

Thread safety: Not thread-safe. Single-threaded use only.

Saves binary data with address header matching ProDOS format.

Writes the provided data to a file. The address and length arguments exist to match BSAVE-style calling conventions.

Current behavior:
