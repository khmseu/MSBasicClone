Detailed Description

Base class for all statement AST nodes.

Statement is an abstract base class representing any executable statement in a BASIC program. Concrete subclasses implement specific statement types (PRINT, IF, FOR, etc.).

All statements must implement the execute() method which performs the statement's action in the context of an Interpreter.
